# 四邊形動畫學習器

一個專為小學生設計的四邊形幾何學習工具，通過互動動畫幫助學生理解四邊形的特性和相互變換關係。

## 生成方式

本程序是由cluade.ai生成的，主要生成過程和提示詞請參考[這裡](https://zxthecoder.com/2025/06/03/%e3%80%8c%e5%9b%9b%e9%82%8a%e5%bd%a2%e6%8e%a2%e7%b4%a2%e3%80%8dclaude-ai-%e7%94%9f%e6%88%90%e9%81%8e%e7%a8%8b/)

## 功能特色

- 🔷 **五種四邊形展示**：正方形、長方形、菱形、平行四邊形、梯形
- 🎯 **互動式屬性探索**：點擊屬性可高亮顯示對應的幾何標記
- 🎬 **動態變換動畫**：觀看四邊形之間的平滑變換過程
- 📊 **實時進度顯示**：動畫過程中的進度條反饋
- 💡 **智能提示系統**：引導式的變換路徑說明
- 🎯 **知識挑戰模式**：15道精心設計的題目，涵蓋四邊形各方面特性

## 文件結構

```

```

## 使用方法

1. **查看形狀屬性**
   - 點擊頂部按鈕選擇不同的四邊形
   - 點擊右側屬性列表高亮對應的幾何特徵

2. **觀看變換動畫**
   - 點擊關係網中的一個形狀作為起點
   - 再點擊另一個形狀作為終點
   - 點擊"觀看變換動畫"按鈕開始動畫

3. **知識挑戰**
   - 點擊"🎯 知識挑戰"按鈕開始答題
   - 回答15道關於四邊形特性的問題
   - 查看詳細的成績分析和錯題復習

4. **互動探索**
   - 動畫過程中可觀察幾何標記的變化
   - 通過進度條了解變換進度

## 題目數據管理

### practices.json 結構說明

題目數據存儲在 `practices.json` 文件中，包含以下主要部分：

- **questionBank**: 題目數組，每道題目包含：
  - `shape`: 相關的四邊形類型
  - `question`: 題目文本
  - `answer`: 正確答案（true/false）
  - `property`: 對應的幾何屬性
  - `explanation`: 詳細解釋
  - `category`: 知識分類
  - `knowledgePoint`: 具體知識點

- **categories**: 知識分類信息
- **difficulty**: 難度級別定義
- **metadata**: 題庫元數據

### 自定義題目

要添加或修改題目，請編輯 `practices.json` 文件：

```json
{
  "questionBank": [
    {
      "shape": "square",
      "question": "正方形的四條邊都相等",
      "answer": true,
      "property": "equal",
      "explanation": "正方形的定義就是四條邊都相等且四個角都是直角的四邊形",
      "category": "sides",
      "knowledgePoint": "正方形的邊長特性"
    }
  ]
}
```

### 錯誤處理

- 當 `practices.json` 加載失敗時，系統會自動使用內置的默認題目
- 加載過程中會顯示loading動畫
- 錯誤信息會以通知形式顯示給用戶

## 技術特點

- **模塊化架構**：HTML、CSS、JavaScript 完全分離，便於維護和擴展
- **純前端實現**：無需服務器，可直接在瀏覽器中運行
- **模塊化數據管理**：題目數據與代碼邏輯分離，便於維護
- **異步數據加載**：支持動態加載題目數據，提升用戶體驗
- **動畫優化**：使用 requestAnimationFrame 實現流暢動畫
- **響應式布局**：適配不同屏幕尺寸
- **容錯機制**：數據加載失敗時的優雅降級

## 部署方式

1. **本地運行**
   ```bash

   ```

2. **Web 服務器**

## 自定義與擴展

### 文件說明
- **index.html**: 頁面結構，包含所有HTML元素和SVG圖形
- **styles.css**: 完整的樣式定義，包括動畫效果和響應式設計
- **script.js**: 所有JavaScript邏輯，包括形狀展示、動畫控制、挑戰模式等
- **practices.json**: 挑戰題目數據庫

### 添加新形狀
可以在 `script.js` 的 `shapes` 對象中添加新的四邊形定義。

### 修改樣式
編輯 `styles.css` 文件來調整：
- 色彩主題
- 動畫效果
- 響應式布局
- 按鈕樣式

### 修改題目
編輯 `practices.json` 文件，添加、刪除或修改題目。

### 調整難度
在 `practices.json` 中的 `difficulty` 部分調整難度分級。

### 功能擴展
修改 `script.js` 來添加新功能：
- 新的動畫效果
- 額外的學習模式
- 更多的交互功能
